# 🌿 Plant Disease Detection System (CNN + Flutter)

A mobile-based plant disease detection system that uses **Convolutional Neural Networks (CNN)** to identify plant diseases from leaf images and provides results through a **Flutter mobile application**.

---

- Randhir K  
- Utsav Thakur  

---

## 📌 Project Overview

This project aims to help farmers and users quickly identify plant diseases using image-based classification. The system combines deep learning with a mobile app interface for real-time predictions.

### 📷 Users can:
- Upload or capture plant leaf images  
- Detect diseases instantly 🤖  
- View disease information and details 📊  

---

## 🧠 Technologies Used

### 🔹 Machine Learning
- Python  
- TensorFlow / Keras  
- CNN (EfficientNet / Custom Model)  
- Google Colab (for training)  

### 🔹 Mobile Application
- Flutter  
- Dart  

### 🔹 Model Formats
- `.keras`  
- `.h5`  
- `.tflite` (for mobile deployment)  

---

## ⚙️ Features
- 🌱 Image-based disease detection  
- 📱 Cross-platform mobile app  
- ⚡ Fast predictions using TFLite  
- 🧾 Supported plant diseases list  
- 🧪 Trained on plant dataset  

---

## 🚀 How It Works
1. User selects or captures an image  
2. Image is preprocessed  
3. Passed into CNN model  
4. Model predicts disease  
5. Result displayed in app
